## 软件功能
- 自动识别同班同学
- 可选的代签生效课程
- 用户白名单机制(防止泛滥成灾)
- 帮助同使用本软件的同班同学签到
- 支持签到码/二维码/手势/位置(可自选)/普通签到
- 支持安卓客户端以及web网页端

## 技术栈
- 大前端: Flutter(Dart)
- 后端: Python(Flask, Pymysql)
- 数据库: Mysql
